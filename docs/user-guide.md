# TUI Desktop Environment - User Guide

## Getting Started

Run `tdesktop` to start the desktop environment. Use `tdesktop --help` for options.

## Navigation

### Keyboard Shortcuts

| Key | Action |
|-----|--------|
| F12 / Ctrl+\` | Toggle command panel |
| Ctrl+Q | Quit desktop |
| Ctrl+L | Refresh screen |
| Ctrl+H | Toggle help panel |
| Alt+1..9 | Switch workspace |
| Alt+Tab | Cycle window focus |
| Alt+F4 | Close focused window |
| Super/Win | Open application launcher |

### Mouse

- Left click: Activate window / press button
- Drag title bar: Move window
- Click [X]: Close window
- Scroll: Scroll content

## Command Panel

The command panel (F12) provides a built-in terminal. Type commands and press Enter to execute. Use Up/Down to browse history.

## Application Launcher

Press Super/Win key to open the launcher. Type to search for applications. Press Enter to launch.

## Configuration

Config file: `~/.config/tdesktop/tdesktop.conf`

Run `tdesktop --reset-config` to restore defaults.
