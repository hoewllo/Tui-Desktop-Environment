# TDESKTOP Key Bindings

## Global
- `F12` / `` Ctrl+` `` - Toggle command panel
- `Ctrl+Q` - Quit desktop
- `Ctrl+L` - Refresh screen
- `Ctrl+R` - Relayout windows
- `Ctrl+H` - Toggle help panel

## Window Management
- `Alt+Tab` - Cycle window focus
- `Alt+F4` - Close focused window
- `Alt+1` through `Alt+9` - Switch to workspace 1-9

## Launcher
- `Super/Win` - Open/close launcher
- `Type` - Filter applications
- `Enter` - Launch selected
- `Up/Down` - Navigate list
- `Esc` - Close

## Command Panel
- `Enter` - Execute command
- `Up/Down` - Browse history
- `Ctrl+U` - Clear input
- `Tab` - Auto-complete
- `Esc` - Close panel
